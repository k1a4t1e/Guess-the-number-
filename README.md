## Description 
Hello! My second project is on this account, I want to introduce you to it.: This is a simple "Guess the number" game written in Python. The tkinter library is used.
## Features 
1. Generate a random number for each new game. 
2. Nice and intuitive graphical interface. 
3. The "play again" button to restart the game without closing the window. 
4. Attempt counter and hints.
